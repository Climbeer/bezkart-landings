# Directives

This directory stores SOP-style directives that define how to execute recurring tasks.

## Naming
- Use action-oriented names like `sync_contacts_to_sheet.md`.
- One directive per repeatable workflow.

## Required sections
Every directive should include:
1. Objective
2. Inputs
3. Outputs
4. Tools/scripts to use
5. Validation criteria
6. Edge cases
7. Constraints
8. Approval requirements

## Usage rule
Before creating a new script in `execution/`, confirm an existing directive and tool cannot solve the task.
